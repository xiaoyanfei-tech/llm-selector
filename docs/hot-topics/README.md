# Daily Hot Topics + Free Solutions

A running digest of public GitHub topics in scope for LLM **selection**, **access/integration**, **cost**, and **deployment**, each with a free, practical solution direction.

Each entry is dated. Topics are sourced from public GitHub issues and abstracted into general problems — no private or personal data.

## Index

- [2026-06-30](2026-06-30.md) — endpoint config fragility, local serving reliability, model switching, multi-provider routing/cost, OpenAI-compatible cost tracking, private RAG reliability

## How this is produced

1. Collect recently active, in-scope public GitHub items (selection / access / cost / deployment).
2. Rank by recency, activity, and scope fit.
3. Write a free, practical solution direction for each.
4. Map each to an `llm-selector` scenario.

## Want a free solution for your own situation?

- Run the CLI: `python selector.py`
- Or open a selection request: https://github.com/xiaoyanfei-tech/llm-selector/issues/new/choose

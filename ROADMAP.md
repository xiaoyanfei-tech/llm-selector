# Roadmap

`llm-selector` is an open-source wedge toward a practical LLM decision platform for OPCs, small teams, and SMEs.

## v0.1 — Selection report foundation

- [x] scenario-based CLI
- [x] model-family shortlist
- [x] Markdown report output
- [x] OPC / SME service workflow
- [x] Chinese README
- [x] examples and sample reports
- [x] issue templates for selection requests

## v0.2 — Feedback and case library

- [ ] share-template and feedback-template workflow
- [ ] anonymized case library
- [ ] model-fit notes by scenario
- [ ] more real-world examples
- [ ] report scoring explanation

## v0.3 — Cost and ROI lens

- [ ] rough token-volume estimator
- [ ] cost profile categories
- [ ] ROI worksheet for OPCs and SMEs
- [ ] PoC success-metric wizard

## v0.4 — Verification integration

- [ ] stronger verify-glm integration
- [ ] endpoint verification checklist in reports
- [ ] generic verify-llm design notes
- [ ] gateway trust/routing risk section

## v0.5 — Web / hosted entry point

- [ ] static landing page
- [ ] web questionnaire prototype
- [ ] downloadable report generator
- [ ] optional anonymized case submission flow

## Long-term vision

Build the decision layer for AI model adoption:

```text
choose the right model → verify the endpoint → run a small PoC → scale only after ROI evidence
```
